# Релизы Android приложения "Список покупок"

## Формат именования файлов
```
shopping-list-{version}-{build-type}-{commit-hash}-{timestamp}.apk
```

## Как установить APK
1. Скачайте APK файл на Android устройство
2. Разрешите установку из неизвестных источников в настройках
3. Откройте файл и следуйте инструкциям установки

## Структура проекта
- **Исходный код:** `../android_studio_project/`
- **Сборка:** `../android_studio_project/app/build/outputs/apk/debug/`
- **Релизы:** `./releases/`

## Версионирование
- **v1.0** - Базовая версия с полным функционалом
- **debug** - Отладочная сборка для тестирования
- **commit-hash** - Короткий идентификатор Git коммита (7 символов)
- **timestamp** - Дата и время сборки в формате ГГГГММДД-ЧЧММ
